Welcome

This mini project behaves like a lightweight Google Drive built for BSc CS demonstrations. Use the auto-generated Swagger UI to try every endpoint, or plug the API into a React/Flutter client.

  • Register a user via /auth/register
  • Log in with OAuth2 password flow at /auth/login
  • Hit /files/upload with a multipart file to push into S3
  • Create expiring public links with /files/{id}/share
  • Inspect overall usage at /admin/summary (admin only)

Stack

FastAPI · SQLAlchemy · AWS S3 · JWT Security

Highlight

Uses real S3 presigned URLs so shared links naturally expire server-side.
